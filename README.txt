我自己编的扫雷游戏

运行方法：
1.安装Consolas字体（Windows可能自带）
2.安装Python
    a.Windows:在python.org上下载python3，运行安装程序。（一定要勾选Add Python To PATH）
    b.macOS:在python.org上下载python3.如果有Homebrew,也可以运行 brew install python3 。
    c.Linux:用软件包管理器安装，比如 sudo apt install python3 等。
3.安装所需库: 终端运行 pip install pygame 和 pip install pygame-textinput 。（如果出错或显示Python 2的停止支持信息，把pip换成pip3）
4.运行:在release中下载source code并解压，在终端中进入该文件夹，运行python Minesweeper.py。（如果出错，把python换成python3）
